## About
This project analyzes insurance claims to understand how demographics, vehicle attributes, and coverage zones influence claim frequency and total claim amounts. The dashboard was built in Power BI to provide interactive insights and support decision-making.

## Files in this repo
- `insurance_data_car.xlsx` — raw dataset (original source: Kaggle)  
- `Insurance_Claim_Analysis.pbix` — Power BI report (if not included, see link below)  
- `dashboard_screenshot.png` — dashboard preview  
- `README.md` — project overview and instructions

## Key Insights
- Private car owners had the highest total claim amounts.  
- Urban zones reported more claims than rural zones.  
- Younger drivers showed higher claim frequency.
